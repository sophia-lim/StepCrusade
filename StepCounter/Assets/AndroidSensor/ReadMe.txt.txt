Android Sensor for Unity3d

To Test this plugin all you have to do is go to Assets/Plugins/Android/SampleAndroidManifest and copy AndroidManifest File to Assets/Plugins/Android
and build the project on actual Android Device and then test the demo

Note: You need actual Physical Android Device to test this Plugin

Also: Pedometer only work on Android 4.4(Kitkat) API 20 for more information look here in this link
http://developer.android.com/reference/android/hardware/Sensor.html#STRING_TYPE_STEP_COUNTER 



How to run the demo

1. go to Assets/Plugins/Android/SampleAndroidManifest copy the Android Manifest file
2. paste it to Assets/Plugins/Android directory
3. go to Asset/AndroidSensor/Scenes and then open the demo scene
4. go to file menu, build settings and then add Demo scene to scenes in build by clicking add current
5. go to file menu, build settings and switch platform to Android
6. go to file menu, build settings and then click player settings and look for other settings 
and then change the bundle indentifier with your bundle indentifier for example: com.mystudio.mygame
7. go to file menu, build settings and then click build and run
8. test the demo
